# 🚄 列车票预订系统（C 语言）
## 📌 项目介绍
这是一个用 C 语言编写的简单的“基于控制台的火车票预订系统”。
它展示了以下内容的使用方法：
* 链表
* 文件输入/输出（二进制读取/写入）
* 基于结构体的数据建模
* 基本用户交互（菜单驱动系统）
该项目被设计成一个以学习为导向的系统，用于练习 C 语言编程和基本的系统设计。
---

## 🧱 特性
* 🔹 添加列车信息
* 🔹 搜索列车（按列车车次或起始城市）
* 🔹 显示所有列车记录
* 🔹 修改列车信息
* 🔹 购买车票
* 🔹 将数据保存至文件
---

### 列车信息

    char startcity[10];
    char reachcity[10];
    char takeofftime[10];
    char receivetime[10];
    int price;
    int ticketnum;
};
```};
```

### 预订信息
```c
结构体 man 包含以下成员：
- char 类型的 num 字段，长度为 10，用于存储用户 ID；
- char 类型的 name 字段，长度为 10，用于存储用户名；
```

---

## 💾 文件存储
* `train.txt` → 存储列车数据（二进制格式）
* `book.txt` → 存储订票记录
⚠️ 仅存储**数据字段**（而非指针），从而确保文件的安全性和可移植性。
---

## 🛠 构建与运行
### 编译（GCC）
```bash
gcc main.c -o train
``````

### 跑步
./train```

> ⚠️ 在 Windows 系统中：
“train.exe”```

## 🧠 重点学习要点
该项目包括：
* 链表的创建与遍历
* 动态内存管理（使用 `malloc` 和 `free` 函数）
* 使用 `fread` 和 `fwrite` 进行安全的文件读写操作
* 通过 `scanf` 进行输入处理
* 基本的系统模块设计
---

## ⚠️ 已知限制条件
* 固定大小的数组（如果不小心操作可能会导致溢出）
* 搜索结果有限（预订中最多显示 10 个匹配项）
* 对无效输入没有数据验证
* 使用了 `conio.h`（仅限 Windows 系统）
